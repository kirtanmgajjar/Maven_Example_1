Problem Statement: Simple cucumber project using datatable

About the Project:
src/main/java has the packages baseClass and utilities
-baseClass package has the BaseClass which contains methods to be performed to start and end the browser, open URL.
-utilities package has the java class for driver setup 
-pages package has the java class for the Home Page and ThankYouPage

src/test/java folder has the package allTest which has the step definition class and the cucumber runner class. 

src/test/resources folder has the BasicContactForm feature file.

reports folder has the cucumber-report.html file which is the report generated after running the tests.

pom.xml contains the dependencies required by the project.
 
testng.xml file is for running the test cases.
